AMQP: Advanced Message Queuing Protocol
高级消息队列协议, 是面向消息中间件提供的开放的应用层协议

RabbitMQ is an AMQP server, message broker消息代理
1. 对于消息的排序、路由(包括点对点和订阅-发布)、保持可靠性、保证安全性
2. AMQP规范了消息传递方和接收方的行为，实现互操作性/HTTP, SMTP, FTP
3. 关注于各种消息如何以字节流的形式进行传递，使用了符合协议的软件之间可以保证消息传递
   It accepts, stores, and forwards binary blobs of data ‒ messages.

RabbitMQ服务器是用Erlang语言(高性能、健壮以及可伸缩性)编写
RabbitMQ实现了"分布式的定时任务调度器"
